# Product Requirements Document (PRD)

## Product
Post-it Board memo web application for personal use.

## Goals
- Fast memo capture and organization by boards.
- Simple visual memo canvas with drag and edit support.
- Reliable auto persistence without explicit save button.

## Primary Users
- Individual users with authenticated sessions.

## Iteration 01 Scope
- Client-side memo board MVP.
- Board and memo CRUD in browser storage.
- Simple editor with completion mode and autosave delay behavior.

## Out of Scope
- Search/sort, file upload, import/export, user profile settings.
