# PRD - SFOOD Memo Board WEB (Iteration 01)

## Objective
Deliver a working frontend that connects to APP API and supports core memo board flows.

## In Scope
- Session auth UI: login, me, logout.
- Board UI: create/list/rename/delete.
- Memo type list and memo CRUD UI.
- API status and error message display.

## Out of Scope
- Google OAuth popup integration.
- Rich text editor toolbar.
- Advanced board tree drag-and-drop.

## Success Criteria
- User can complete core flow against APP API without direct API tool usage.
- API integration respects APP response/error contract.
