# Done Criteria (Web)

## 필수 완료 조건
- 계획/요구사항/아키텍처/PRD/API 스펙 문서가 최신 상태다.
- 구현 범위 기능이 PRD 체크리스트 기준으로 완료 처리되었다.
- 테스트 또는 스모크 검증 결과가 기록되었다.

## 품질 게이트
- 무세션/만료 세션(401) 처리 동작이 일관된다.
- 자동저장 실패 시 사용자에게 오류가 노출된다.
- API 에러 코드별(400/401/404/409) 처리 기준이 문서/코드와 일치한다.
- 사용자 정보/로그아웃 버튼이 LNB 하단에 배치된다.
- 보드 이동(드래그 앤 드롭) 테스트가 필수 통과한다.
- 줌 퍼센트 표시와 수동 줌 버튼이 동작한다.
- 화면 선반영 -> API 호출 -> 실패 롤백 흐름이 구현/검증되었다.
