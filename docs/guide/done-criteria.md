# Done Criteria (WEB)

## Mandatory
- Iteration plan document exists.
- Code matches planned scope.
- Test result document includes commands and pass/fail status.
- Iteration result and change log documents are written.

## Quality Gates
- Session-based API calls include credentials.
- Required user flows are runnable from the UI.
- API error messages are visible in UI.
- Layout is usable on desktop and mobile.
